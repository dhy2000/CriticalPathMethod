# 经济管理 CPM-关键路径法

## 输入数据格式

第一行两个整数 `n`, `m`, 为事件个数和关系个数。

接下来 `n` 行为事件信息, 每行一个字母和一个整数, 为该事件的标号和持续时间。格式形如 `A 1`。

接下来再有 `m` 行为事件之间的连接关系。如果两件事之间没有时间约束则格式为 `A B 0`, 如果有约束则格式为 `A B FS 3`。事件之间的时间约束包括 `FS`, `FF`, `SS` 和 `SF` 四种。

## 输出数据格式

若干个 3x3 矩阵，每个矩阵为一个事件的信息。

最后为关键路径。
